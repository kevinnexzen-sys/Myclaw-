# Myclaw v1.0.0 - Mobile Release

🎉 First Mobile Release of Myclaw!

📱 Download Options:
- Android APK (ARM64) - Recommended for modern Android devices
- Android APK (ARMv7) - For older Android devices  
- Android APK (x86_64) - For emulators

🚀 Features:
✅ Voice-first AI Assistant
✅ Autonomous Code Generation
✅ Real-time Windows PC Control (Wake-on-LAN)
✅ Sub-agent Delegation
✅ Multi-device Sync via Supabase
✅ Background Monitoring
✅ Task Learning & Evaluation

📋 System Requirements:
- Android 8.0 or higher
- ~150 MB free storage
- Internet connection

📖 Installation Guide:
See INSTALLATION.md for detailed setup instructions.

🔗 Build from Source:
See SETUP_BUILD.md for development build instructions.
